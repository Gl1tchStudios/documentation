## **Overview**

House Robbery is a sophisticated criminal progression system designed specifically for FiveM roleplay servers. This system provides realistic break-in scenarios, dynamic loot distribution, and comprehensive criminal career advancement. With support for interior shell spawning, intelligent noise detection, progressive difficulty scaling, and phone-based job notifications, House Robbery transforms basic theft mechanics into an immersive, risk-versus-reward criminal experience with meaningful character progression.

### **Features**

* **Dynamic Interior System**: Procedurally spawned shell-based interiors with multiple searchable locations and realistic house layouts
* **Progressive XP System**: 75-level advancement system with scaling XP requirements and time reduction rewards for experienced criminals
* **Six-Tier Loot System**: From common household items to legendary artifacts
* **Lockpicking Mechanics**: Realistic entry system with item consumption, break chances, and police alert triggers on failed attempts
* **Intelligent Alert System**: Level-based noise detection with configurable thresholds and delayed police notifications for balanced gameplay
* **Phone Integration**: Seamless lb-phone compatibility with contact-based job notifications and immersive communication
* **Skill-Based Progression**: Tier unlock system tied to player level, reducing search times and alarm risks as criminals gain experience
* **Risk Management**: Strategic noise alarm system where higher-level players can operate more quietly and avoid detection
* **Fully Configurable**: Extensive configuration for loot tables, XP scaling, timer reductions, tier chances, and difficulty progression
* **Dynamic Rewards**: Level-scaling loot system with multi-item searches and weighted probability distribution
* **Duty System**: Professional sign-in/sign-out mechanics with duty status tracking and job availability management
* **Discord Logging**: Complete audit trail with webhook integration for administrative oversight

## **Installation Requirements**

### **Dependencies**

* **ox\_lib**: Modern UI framework for menu systems and notifications and more
* **ox\_target**: Standalone "third-eye" targeting resource.
* **ox\_inventory**: Slot-based inventory with metadata.

**Shell Dependencies**

This resource requires interior shell resources to function properly. You must have at least one of the following shell packs installed:

* **lynx\_shells:** A comprehensive collection of pre-built interior shells for various property types
* **K4MB1-StarterShells:** A starter pack of MLO shells providing basic interior environments

**Phone Integration**

This resource is designed to work with **lb-phone** out of the box for job notifications and communication features. However, lb-phone integration is **not strictly required** for the script to function.

The phone-related functions are located in the open source files and can be easily modified to work with your preferred phone resource.

For more help regarding if your phone system works with our script, feel free to open a ticket in our discord.
